# Product Context: Erick Larsen Personal Website

## Problem Statement

Frontend developers need a professional online presence to showcase their skills, experience, and projects to potential employers, clients, and the developer community. A personal website serves as a living portfolio that demonstrates both technical capabilities and design sensibilities.

## Solution

A modern, performant personal website built with Next.js that:

- Presents professional information in an engaging format
- Demonstrates frontend development expertise through the site itself
- Provides easy access to social profiles and project repositories
- Supports both light and dark themes for user preference

## User Experience Goals

### First-Time Visitors

- Immediately understand who Erick is and what he does
- Quickly scan experience and expertise
- Easy access to social links and contact methods
- Smooth navigation between sections

### Returning Visitors

- Check for new experiments/projects
- Read blog posts (future feature)
- Theme preference remembered

## Content Strategy

### Home Page

- Brief introduction with personality ("Hey what's up!")

- Experience timeline showing career progression
- Technology expertise icons organized by category:
    - Platform/Frameworks: Next.js, React, Redux, TypeScript, Node.js, Svelte, PHP, Express, Bun
    - Styling/Database: Tailwind, Styled Components, Sass, MySQL, PostgreSQL, MongoDB
