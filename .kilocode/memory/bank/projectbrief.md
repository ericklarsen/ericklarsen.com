# Project Brief: Erick Larsen Personal Website

## Overview

A personal portfolio website for Erick Larsen, a Frontend Developer. The website showcases professional experience, technical expertise, and personal projects/experiments.

## Core Purpose

- Present professional identity and brand as a Frontend Developer
- Display work experience and career progression
- Showcase technical skills and expertise
- Share experiments and side projects
- Provide a platform for future blog content
- Enable visitors to connect via social media links

## Target Audience

- Potential employers and recruiters
- Fellow developers and tech professionals
- Clients seeking frontend development services
- Anyone interested in Erick's work and projects

## Key Features

1. **Home Page**: Hero section with introduction, experience timeline, and expertise showcase
2. **About Page**: Detailed information about the developer
3. **Experiments Page**: Portfolio of side projects with GitHub links
4. **Blogs Page**: Platform for future blog content (currently placeholder)
5. **Dark/Light Mode**: Theme toggle with system preference detection
6. **Responsive Design**: Mobile-first approach with adaptive layouts

## Design Philosophy

- Clean, modern aesthetic with professional appearance
- Minimalist UI with focus on content
- Smooth user experience with theme persistence
- Accessible and performant static site

## Success Metrics

- Fast page load times (static export)
- SEO-friendly structure
- Consistent branding across all pages
- Easy navigation and content discovery
